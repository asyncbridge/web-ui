Object Detection. Detect the crested gecko.

The project name is "Lilly White". It provides web based object detection service for crested gecko. The crested gecko has various morphs and traits. It is difficult for the beginner of breeder to classify morphs and traits of the crested gecko. It helps them to understand it easily through this service.

Object Detection. Detect the Tsinghua-Tencent 100K.

This is object detection service which is trained by Tsinghua-Tencent 100K. The Tsinghua-Tencent dataset consists of Chinese traffic-sign classes. It uses FPN-CNN model which is proposed by my paper for object detection. I deployed Caffe deep learning framework and used GPU computing for object detection based on Docker. The model files are downloaded from AWS S3 whenever starting Docker container service.
